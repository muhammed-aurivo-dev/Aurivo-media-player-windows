# Katkı Rehberi (CONTRIBUTING)

Teşekkürler! Aurivo Media Player’a katkı yapabilirsiniz.

## Hızlı akış
1. Issue açın (bug/feature).
2. Branch oluşturun: `feat/...` veya `fix/...`
3. Değişikliği küçük ve odaklı tutun.
4. PR açın ve açıklamaya test adımlarını yazın.

## Lokal geliştirme
- Bağımlılıklar: Node.js (LTS), Git
- Kurulum: `npm ci`
- Native addon: `npm --prefix native ci`
- Çalıştırma: `npm run start`

## Build (özet)
- Windows: `npm run build:win`
- Linux: `npm run build:linux`

## Kod stili
- Mevcut dosya yapısını ve isimlendirmeyi bozmayın.
- UI değişikliklerinde ekran görüntüsü ekleyin.

## Güvenlik
Güvenlik açığı bulursanız public issue açmak yerine önce maintainer ile iletişime geçin.
